# Homify – Full Stack Airbnb-Like Rental Platform

Homify is a full-stack web application inspired by Airbnb that allows users to browse, list, and manage rental properties. The project is built using the MERN stack and follows the MVC architecture with secure authentication and authorization.

## Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript (ES6+)

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Mongoose)

**Authentication & Security**
- JWT (JSON Web Tokens)
- Cookies
- Password hashing

**Architecture**
- MVC (Model–View–Controller)
- RESTful APIs

## Features

- User authentication and authorization (JWT + cookies)
- Role-based access control (User / Admin)
- Property listing with full CRUD operations
- Secure backend using MVC architecture
- RESTful API integration between frontend and backend
- Responsive UI for different screen sizes
- Form validation and error handling

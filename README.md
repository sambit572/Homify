# Homify – Full Stack Airbnb-Like Rental Platform

Homify is a full-stack web application inspired by Airbnb that enables users to browse, list, and manage rental properties. It is built using the MERN stack and follows the MVC architecture with secure authentication and authorization mechanisms.

---

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Authentication & Security
- JWT (JSON Web Tokens)
- Cookies
- Password hashing

### Architecture
- MVC (Model–View–Controller)
- RESTful APIs

---

## Features

- User authentication and authorization using JWT and cookies
- Role-based access control (User / Admin)
- Property listing and management with full CRUD functionality
- RESTful APIs for frontend–backend communication
- MVC-based backend structure for scalability
- Responsive UI across multiple screen sizes
- Form validation and robust error handling
